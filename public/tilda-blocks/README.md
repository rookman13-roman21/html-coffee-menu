# Tilda HTML-блоки — CaféDesk

Каждый файл — отдельный HTML-блок для вставки в Tilda через «Блок HTML/JS».

## Порядок блоков на странице

| Файл | Содержимое |
|---|---|
| `01-hero.html` | Главный экран — заголовок, подзаголовок, CTA-кнопка |
| `02-benefits.html` | Преимущества — 6 карточек с иконками |
| `03-features-detail.html` | Детальные фичи — рецептуры, финмодель, техкарты |
| `04-for-whom.html` | Для кого — портрет целевой аудитории |
| `05-stats.html` | Цифры и факты — 30+ рецептов, 0 ₽ и т.д. |
| `06-pricing.html` | Тарифы — Бесплатно vs Команда |
| `07-cta-final.html` | Финальный призыв к действию |

## Как вставить в Tilda
1. В редакторе страницы добавить блок → «Другое» → «HTML»
2. Вставить содержимое нужного файла в поле кода
3. Сохранить и опубликовать
