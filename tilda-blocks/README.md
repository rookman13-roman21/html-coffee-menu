# Tilda HTML-блоки — CaféDesk

Каждый файл — отдельный HTML-блок для вставки в Tilda через «Блок HTML/JS».  
Все блоки содержат inline `<style>` и `<script>` — никаких внешних зависимостей.

## Особенности hero-блока (01-hero.html)

- **Макет приложения** — правая колонка с интерактивными вкладками: Дашборд / Себестоимость / Финмодель
- **Авто-ротация** — вкладки переключаются каждые 3.2 сек с прогресс-баром; пауза при наведении
- **Анимации появления** — fade+slide-up (текст) и fade+scale (макет) с stagger по дочерним элементам

## Как вставить в Tilda
1. Открыть редактор страницы → добавить блок → «Другое» → «HTML»
2. Вставить содержимое нужного файла целиком
3. Сохранить и опубликовать

## Деплой

Папка `tilda-blocks/` дублируется в `public/tilda-blocks/` — Vite автоматически копирует `public/` в `dist/`.

⚠️ При изменении файлов в `tilda-blocks/` — обязательно синхронизировать:
```bash
cp tilda-blocks/NN-name.html public/tilda-blocks/NN-name.html
```
